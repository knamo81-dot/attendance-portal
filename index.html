근태 포털 분리형 구조

기존 attendance.html을 유지한 채 기능별 진입 페이지를 먼저 분리한 구조입니다. 업로드와 분석 디버깅을 서로 덜 건드리게 만드는 1단계 분리본입니다.

기존 로직 기반

근태관리

출퇴근 상세, 필터, 결근확인/OT확인, 사유 저장 화면.

분석대시보드

KPI, 차트, 근태 요약 등 전체 현황 확인.

심층분석

심층분석 내부 4개 기능을 하나의 화면으로 유지.

출퇴근 누락 분석

출근누락/퇴근누락 순위와 조치용 분석 화면.

관리자

ERP/세콤 업로드, 월삭제, 업로드 현황 전용 화면.

서버 연결 문제는 이 화면을 기준으로 먼저 안정화하는 것이 좋습니다.
