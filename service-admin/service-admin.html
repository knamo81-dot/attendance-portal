서비스 관리자

회사 등록, 회사별 관리자 계정, 사용 앱 설정을 관리하기 위한 전용 공간입니다. 현재는 2단계 기본 화면이며, 다음 단계에서 Supabase 연동 기능을 붙입니다.

회사 관리

회사 등록, 회사명 변경, company_code 확인, 활성/비활성 상태를 관리합니다.

회사별 관리자

각 회사의 최초 관리자 계정 생성과 company_id 연결을 관리합니다.

앱 사용 설정

근태, 폐수, 시약·초자, 인력운영 등 회사별 사용 앱을 켜고 끄는 영역입니다.

다음 단계에서는 companies, users, 회사별 앱 설정 테이블을 연결해 실제 등록·수정 기능을 추가합니다.
